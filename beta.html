 NOCTURNE ALCHEMY PLATFORM

Welcome to the BETA page for the Nocturne Alchemy Platform – a flexible eurorack module series that redefines versatility and creativity. Each module within the Nocturne Alchemy Platform shares the same robust Arduino-based hardware, allowing you to effortlessly swap functionalities through our intuitive web firmware loader (See the video on this page) By purchasing one module, you gain access to the full range of available firmwares, including the Slight Of Hand, Arp Of Darkness, Shroud Of Turing, and the Seventh Summoner as well as others that are planned for future release!

This is a firmware uploader for Nocturne Alchemy Platform  modules such as Slight Of Hand and Arp Of Darkness. It only works in Chrome on a Mac or PC. You will need a USB-A to USB Mini cable. If you don't have one laying around, you can find many inexpensive ones on Amazon

Just a quick word of warning: If you are here it's likely that you are a pro, and not afraid to beta test unproven firmware. Obviously, things could go sideways and weird things could happen like having your calibration go wonky, or a black hole suddenly appearing in your studio sucking in all you hold dear. I personally test firmware for some time before letting it slide to beta, and I try to take every precaution but you know... "at your own risk" and all that.      

Read through all the steps below before you update the firmware.

  1. Make sure to unplug your module from the Eurorack power cable first.
  2. Plug in your USB cable to the mini USB port on the bottom of your module
  3. The chip next to the USB port will have an "O" or an "N" on it - Make note of this.
  4. If you don't have an N or O on the chip, try "O" first, then "N". Guessing won't hurt anything. (The update will just fail. If so, choose the other one. It has to be one of them.)
  5. Click on the button for the firmware you wish to update with the correct letter on it. 
  6. Select the USB serial option from the prompt that pops up. (Sometimes, when other USB devices are plugged in it take a few tries - just unplug, replug, and try again)
  7. Wait until the uploaded finishes (100%)
  8. You're done. Unplug from the USB cable, plug in the power, and go make some music.

Arp Of Darkness


V1.4.0 BETA : 03/27/2026 

This is a BETA firmware for the arduino-based Arp Of Darkness. 

New Features:

  • Remix Buffer - Randomly shuffles the note order in the buffer on the next clock. Note values are preserved — only the playback order changes.
  • Auto-Remix Modes- Automatically reshuffles the buffer every 2, 4, or 8 loops.
  • Slight Of Hand Mode  - Allows the user to boot into a CV Keyboard mode that functions like the Slight Of Hand module by simply holding the Shift/High C button during power on and boot.

Bug Fixes & Enhancments

  • Fixed bug that ripped a small hole in the time-space continuum that may or may not end everything in existence as we know it... again. (Oops?)
  • Firmware 1.4.0 User Manual (With Firmware Flashing Instructions)

 

Seventh Summoner


V1.1.0 BETA: 08/05/2025 

This is BETA firmware for the arduino-based Seventh Summoner. 

New Features:

  • New "Slight Of Hand" mode - Allows the user to boot into a CV Keyboard mode that functions like the Slight Of Hand module by simply holding the Shift/High C button during power on and boot.  

Bug Fixes & Enhancments

  • Fixed a critical bug where in rare occurances calibration data could be corrupted and zeroed out due to EEPROM protected address being overwritten resulting in user having to recalibrate the module 
  • Firmware 1.1.0 User Manual (With Firmware Flashing Instructions) 

 

 

Shroud Of Turing


V1.2.4 BETA: 05/01/2026 

This is BETA firmware for the arduino-based Shroud Of Turing. 

New Features:

  • Added 5 state save/load slots mapped to black keys (C#/D#/F#/G#/A#) 
    •  Octave Down + black key saves full module state to EEPROM slot 
    •  Octave Up + black key loads state from EEPROM slot 
    • Each slot stores: shift register, step length, voltage range, pot value, scale, and rotation offset (12 bytes per slot) 
    • Pot pickup behavior after load prevents jarring probability jumps

 

Bug Fixes & Enhancements:

  • The rotate forward and rotate backward functions have been optimized to function more how the user expects with the post-quantized values being used for the rotation instead of shift register values.
  • Fix a bug that in some cases could cause the reset to trigger double notes on the first note of the reset  
  • Firmware 1.2.4 User Manual (With Firmware Flashing Instructions)
